# Scales4Dummies

Интерактивный гриф гитары в браузере: настраиваемый строй по струнам, 6/7 струн, тональности с подсветкой нот, светлая и тёмная тема. Инструмент для гитаристов, которые не знают ноты: прогрессии по ступеням.

**Сайт:** [sorokafoto.github.io/Scales4Dummies](https://sorokafoto.github.io/Scales4Dummies/)

---

На одном экране: настройки строя и тональности, панель **прогрессии по ступеням** (кнопки ступеней, цепочка аккордов, «Очистить») и гриф с подсветкой нот тональности и нот выбранного аккорда в цепочке.

---

## Возможности

- **Строй** — для каждой струны отдельный выбор ноты (от толстой к тонкой). Кнопки ▲/▼ сдвигают строй всех струн на полтона вверх или вниз.
- **6 / 7 струн** — переключатель; по умолчанию Standard E (6) и B Standard (7).
- **Тональность** — тоника (C, C#, … B) и тип: Major, Natural Minor, Harmonic Minor, Major/Minor Pentatonic, Dorian, Phrygian, Lydian, Mixolydian, Locrian.
- **Подсветка нот** — на грифе подсвечиваются только ноты выбранной тональности; ноты вне тональности отображаются приглушённо.
- **Ступени** — под грифом показываются ноты тональности и соответствующие ступени (латинские цифры с бемолями/диезами: I, bII, bIII и т.д.).
- **Прогрессия** — кнопки ступеней тональности; клик добавляет аккорд в цепочку; клик по аккорду в цепочке подсвечивает его ноты на грифе; «Очистить» сбрасывает цепочку.
- **Hover** — при наведении на ноту из тональности подсвечиваются красным все такие же ноты на грифе.
- **Тема** — переключатель «День / Ночь» (тёмно-серый фон для игры ночью).
- **Сохранение** — строй, тональность и тема сохраняются в `localStorage`.

## Запуск локально

Откройте `index.html` в браузере (файлом или через локальный сервер).

---

## Публикация на GitHub

1. Создайте репозиторий на GitHub (например `Scales4Dummies`).
2. Залейте в него файлы: `index.html`, `styles.css`, `app.js`, папку `docs/` (при необходимости), `.nojekyll`, `README.md`.
3. В репозитории: **Settings → Pages**.
4. **Source**: Deploy from a branch.
5. **Branch**: `main` (или `master`), папка **/ (root)**.
6. Сохраните — через минуту сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

Сайт статический, всё работает в браузере; для GitHub Pages сборка не нужна.
